Análise e Desenvolvimento de Sistemas

Instituição - Uninter

Este curso de graduação da Uninter tem como objetivo formar profissionais qualificados para atuar no desenvolvimento, manutenção e gestão de sistemas de software. O currículo inclui disciplinas como programação, engenharia de software, banco de dados, redes de computadores e segurança da informação. Os alunos também aprendem a utilizar metodologias ágeis e a aplicar práticas de análise e modelagem de sistemas.

Portfólio

Instituto Da Mente

O projeto "Instituto da Mente" é uma plataforma web que desenvolvi para um instituto voltado à saúde mental. A aplicação inclui seções para serviços clínicos, como psicanálise e hipnoterapia, além de cursos para profissionais da área. O objetivo era fornecer uma experiência de usuário fluida, facilitando o acesso às informações e serviços oferecidos pelo instituto.

Rocketnotes

O projeto "Rocketnotes" é uma aplicação web desenvolvida para gerenciamento de notas pessoais. A interface é simples e direta, permitindo que os usuários criem, visualizem e organizem suas notas de forma eficiente. O design é responsivo e minimalista, focando na usabilidade.

Web Development - Full Stack

O projeto "World Cup Calendar" é uma aplicação web que desenvolvi para exibir os jogos da Copa do Mundo de 2022. Utilizei o Figma apenas para criar o wireframe, e todo o código foi implementado por mim. O objetivo do projeto é oferecer uma interface intuitiva e responsiva para que os usuários possam acompanhar o calendário dos jogos de forma prática e visualmente atraente.
